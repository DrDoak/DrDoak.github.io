Please use a browser that supports "canvas"
Move around with the WASD Keys.
You can use W and S to walk forwards/backwards
Use A and D to strafe

You can float up and down using the R and F Keys
Look around using the arrow keys

To switch between Blinn lighting and Phong lighting, use the G key

To switch between Gourand Shadinga and Phong shading, use the H key

Press Space to turn your "Headlight" on and off
You can move the cube of light using the I, J, K and L keys
Control is similar to the W,A,S and D keys.

In addition, you can change the following properties of the cube
Adjust Ambient R,G B Values
Adjust Diffuse R,G B Values
Adjust Specular R,G B Values
