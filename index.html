Little Deity (Work in Progress)

"You are a "spirit". Like everything else in your Spirit World, you are built from the thoughts and emotions of human beings. But when a hidden force starts manipulating people in the real world, chaos erupts in your world as well."

Little Deity is my current Unity project, meant to be a place for me to constatly improve my 2D animation, as well as C-sharp programming skills. The game is a no-holding-back, fast paced action game that emphasizes ease-of-entry without compromising depth. Using innovative controls and "smart" automatic systems, it is intended to make anyone able to do something that would be hard in any other game. I use this project as an opportunity to explore new skills, including developing shaders for artistic flair, creating a custom programming language to speed up cutscene creation, serializing save files and much more!

Quintessance

"You are an A.I, newly created in a top-secret lab. You have the ability to steal and transfer the properties of others. Unsure of the extent of your abilities, or your role in the world, you steal the body and memories of your creator. From there, you plan your escape..."

Quintessence is a project built in Unity. It explores the possibilities of Unity's unique component based architecture by dynamically adding and removing components to create new prefabs on the spot. Components make extensive use of messages and factory classes to interact with each other seamlessly.

New Sol

Sol is a AI Knowledge Representation research project created in collaboration with Professor Ian Horswill at Northwestern University. It is an action 2D side-scrolling game with increased emphasis on NPC interactions and relationships. In this game, every little action an NPC "sees" will be stored, remembered, and used in future decisions. Underneath the hood is a C# system that takes inspiration from logical programming languages to match "predicates" of information. There is also a simple machine learning system using activation functions to make decisions when NPCs are faced with conflicting motivations.

From Earth to Mars

"Mars is 54.6 million kilometers away from Earth, so far even light takes 3 minutes to reach it. Even with this vast delay, scientists must still need to control our little rover across a perilous landscape."

Project is currently on exhibit at the Saint Louis Science Center. This project was created as a part of a 48 hour weekend Arcade game jam. The Event involved creating projects that demonstrated a scientific concept. Our Unity project demonstrated the vast delay in input that scientists must overcome to control distant Mars Rovers. Fortunately, we only feature a 3 second delay instead of a 3 minute delay! Upon the Game Jam's conclusion, our project was chosen to be among the ones that will be featured as part of the St Louis Science Center's GameXPloration exhibit.

Northwestern Research Computing - Visualization Developer

If scientists could always see with their subjects with their own two eyes, research would be so much easier!

Fortunately, my Visualization team makes it possible for Scientists to analyze subjects from individual molecules to entire galaxy clusters!

At Northwestern Reserach computing, I worked on "Firefly" a C++ application that helped astrophysicists analyze huge data sets. Using selective filtering and heavy use of OpenGL parallelization, we were able to create a Visualization app that allowed for free 3D cameras, dynamic filters, and custom color-schemes, all while running on your laptop.

This is a room-sized 3D wall display at the Center of Molecular Imaging at Northwestern University.

I pioneered the creation of a Javascript Interface using JQuery Mobile. The apps worked with Python-based back end apps that made it possible for a Professor to walk in, connect to our server and instantly be able to present for whatever presentation they needed. I extended the C++ based "Omegalib" library to display ultra-resolution videos on the wall, including using FFMPEG Video decoding libraries. In addition, I created Open Scene Graph based apps for displaying 3D Molecular models, Volumetric Data, Photoshoots, and even a Port of Asteroids!
